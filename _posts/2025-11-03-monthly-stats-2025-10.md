We believe transparency is important to build a trustworthy open-source community, so we regularly share our usage and sustainability statistics.

Below are some statistics we found interesting (in parentheses, you can see the figures for the previous month). If you are interested in any other figures, please let us know.

## Detailed Statistics for October 2025

### Usage

* Website & Docs visitors: **9.2K** (10K)
* Website & Docs page views: **49K** (55K)
* Top 5 Docs pages: **Quickstart, Docs Home, Gherkin reference, SpecFlow Migration, IDE Setup** (Docs Home, Quickstart, SpecFlow Migration, Gherkin reference, IDE Setup)
* NuGet package downloads (Reqnroll package, last 6 weeks): **1.83M** (1.7M)
* NuGet package downloads compared to SpecFlow NuGet downloads (based on "per day average"): **77%** (71%)
* Number of Reqnroll projects loaded by VS2022 extension: **407K** (404K)
* Number of Reqnroll projects loaded by Rider extension: **8.3K** (10K)
* Upgrades to the latest VS2022 extension: **8.7K** (May: 7.6K)
* Installations of VS2022 extension (all versions, without upgrades): **76.4K** (N/A)
* Top 3 countries (from VS2022 extension usage): **USA, India, United Kingdom** (India, USA, United Kingdom)
* Reqnroll projects (.NET projects executed by Reqnroll): **23K** (22K)
* Reqnroll test executions: **12.5M** (11.4M)
* Reqnroll test executions per project per month: **avg: 538, median: 56** (avg: 515, median: 53)
* Ratio of CI/CD executions: **45%** (45%)
